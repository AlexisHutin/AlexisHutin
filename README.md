## [Homelab](https://github.com/AlexisHutin/home_lab)

Self-hosted platform and experimentation environment.

A multi-node setup built around Docker, with a growing separation between application and infrastructure layers.

Includes:
- media stack (Plex ecosystem)  
- AI stack (Ollama, local assistants, speech-to-text)  
- observability stack (Prometheus, Grafana, InfluxDB)  
- automation (n8n)  
- infrastructure services (AdGuard, monitoring node, backups in progress)  

Used as a playground to experiment with:
- Infrastructure as Code and GitOps approaches  
- service architecture and isolation  
- monitoring and alerting strategies  
- evolution toward more platform-oriented patterns
